# Theseus Rover

Low-cost, open-source rover platform for education, research, and DIY robotics, 6 wheeled rover design is based on existing rovers like ESA's Rosalind Franklin (ExoMars) and CSA's MESR and REX for planned exploration of Martian surface.

This design was inspired by NASA's JPL open source rover (https://github.com/nasa-jpl/open-source-rover), but can be build with parts widely available worldwide and 3D printing, which reduced cost of minimal working version by half including cost of basic 3D printer.

Total cost of the rover is just under $800 or £600 including cost of 3D printer. 

Triple bogie suspension, compared to rocker-bogie provides comparable capabilities, but allows for much simpler design.
